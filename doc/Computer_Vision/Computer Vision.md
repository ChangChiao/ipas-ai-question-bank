# Computer Vision

## 📝 概念解釋 (Concept Explanation)
電腦視覺 (Computer Vision, CV) 是一門人工智慧的子領域，旨在使電腦和系統能夠從數位影像、影片或其他視覺輸入中提取有意義的資訊，並基於這些資訊採取行動或提供建議。與人類擁有眼睛與大腦能理解所見事物類似，電腦視覺透過攝影機捕捉影像，並交由深度學習等演算法來解析內容與模式，達到「看見」並「理解」的目的。

## 💡 使用場景 (Use Cases)
- **自動駕駛與輔助駕駛系統 (ADAS)**：車輛利用攝影機即時辨識道路標線、行人、車輛與交通號誌，實現自動變換車道或緊急煞車。
- **醫療影像分析**：協助醫生判讀 X 光片、MRI 或 CT 影像，自動偵測腫瘤或異常病灶，提高早期診斷的準確度。
- **工業瑕疵檢測**：在智慧製造產線上，透過高速攝影機與電腦視覺自動找出產品表面的刮傷或組裝瑕疵，大幅降低人工品檢成本。
