# Decision Tree (決策樹)

> **📝 概念解釋：**
> 決策樹 (Decision Tree) 是一種監督式學習演算法，外觀如同一棵倒置的樹形結構。它藉由計算資訊增益 (Information Gain) 或 Gini 不純度等指標，在每一個節點上挑選最具區分能力的特徵來切割資料集，直到最後的葉節點得出預測分類結果或數值。它的過程完全可視化且解釋性極高。
>
> **💡 使用場景：**
> 經常應用在醫療輔助診斷，因為醫生需要明確的推理軌跡；或者是企業內部的人事決策、客戶流失判定等需要向非理工背景主管解釋模型判斷邏輯的情境。
>
> **🔗 關聯關鍵字：**
> `Information Gain`, `Gini Impurity`, `Overfitting`, `Random Forest`
