# Logistic Regression (邏輯迴歸)

> **📝 概念解釋：**
> 羅吉斯迴歸 (Logistic Regression) 雖然名字有「迴歸」，但它實際上是一種分類演算法 (通常用於二元分類)。它將線性迴歸的輸出結果通過 Sigmoid 壓縮函數，將數值轉換成 0 到 1 之間的機率值，大於特定閾值 (如 0.5) 則判斷為正類，否則為負類。
>
> **💡 使用場景：**
> 被廣泛用於預測「發生或不發生」的二分法情境。最經典的應用為電子信箱的垃圾郵件分類、醫療檢測(如預測腫瘤是惡性還是良性)以及預測會員是否會流失。
>
> **🔗 關聯關鍵字：**
> `Classification`, `Sigmoid Function`, `Binary Classification`, `Cross Entropy`
