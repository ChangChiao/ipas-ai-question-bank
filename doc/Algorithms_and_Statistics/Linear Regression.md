# Linear Regression (線性迴歸)

> **📝 概念解釋：**
> 線性迴歸 (Linear Regression) 是一種監督式學習演算法，用於尋找自變數 (X) 與連續型應變數 (Y) 之間最佳的線性關係。演算法透過最小化「預測值與實際觀察值之間的誤差平方和 (MSE)」來找出一條最佳擬合的直線。
>
> **💡 使用場景：**
> 廣泛運用在趨勢預測。比如房地產界可使用坪數、距離市中心的距離做為自變數，預測房屋的成交價格；或用於銷售額的長期趨勢預測。
>
> **🔗 關聯關鍵字：**
> `Supervised Learning`, `MSE (Mean Squared Error)`, `Gradient Descent`, `Regression`
