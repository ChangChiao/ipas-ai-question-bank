# Classification and Regression

## 📝 概念解釋 (Concept Explanation)
分類 (Classification) 與迴歸 (Regression) 是監督式學習中最常見的兩大任務類型。兩者的核心差異在於「預測輸出的目標變數類型」：
- **分類 (Classification)**：目標是預測「離散型的類別標籤」。模型會將輸入資料分配到預先定義好的特定類別中 (例如：是/否、貓/狗/鳥)。
- **迴歸 (Regression)**：目標是預測「連續型的數值」。模型會根據輸入的特徵計算出一個具體的實數 (例如：價格、溫度、機率)。

## 💡 使用場景 (Use Cases)
- **分類場景：垃圾郵件偵測**：將收到的電子郵件標記為「正常信件」或「垃圾信件」。
- **分類場景：醫療診斷**：根據病患的各項檢測指標，判斷其是否罹患某種疾病 (例如糖尿病的陰性/陽性)。
- **迴歸場景：房價預測**：根據房屋的坪數、屋齡、地點等特徵，預測該房屋的潛在市場售價 (例如 1500 萬)。
- **迴歸場景：股市趨勢或銷量預測**：預測未來的連續性數據，如某商品下個月的銷售額。
