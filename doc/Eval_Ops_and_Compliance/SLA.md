# SLA (服務水準協議)

> **📝 概念解釋：**
> 服務級別協定 (Service Level Agreement, SLA) 是一份由服務供應商與客戶之間簽署的正式合約。它將服務的各項具體標準與品質目標數據化並白紙黑字訂定，例如保證系統可用性要達到 99.9%、回應時間(Latency)在 200 毫秒內等。若未能達標，供應商通常需要給予客戶相對應的賠償或違約金。
>
> **💡 使用場景：**
> 企業導入雲端 LLM 服務 (如 Azure OpenAI) 時的關鍵採購考量。因為大型語言模型運作容易遇到系統擁擠，合約中的 SLA 會保證企業專案能穩定且不受影響地上線運作。
>
> **🔗 關聯關鍵字：**
> `Availability`, `Downtime`, `Cloud Computing`, `QoS`
