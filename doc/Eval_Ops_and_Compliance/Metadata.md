# Metadata (元數據)

> **📝 概念解釋：**
> 詮釋資料或元數據 (Metadata) 簡言之就是「關於資料的資料 (Data about data)」。它被設計用來描述資料背後的關聯資訊以方便管理與檢索。例如，一張數位照片本身是資料，但記錄在照片檔案內的拍攝日期、相機機型、GPS 座標及光圈大小等，就是它的 Metadata。
>
> **💡 使用場景：**
> 在 RAG (檢索增強生成) 系統架構中非常關鍵。為了避免 AI 在大海撈針中找錯向量知識，每份切片(Chunks) 上都會貼上如「部門來源、創建時間、標籤」等 Metadata。使用者提問時，系統可先針對 Metadata 進行硬性過濾（如限定只搜『今年度』文件），再進行向量相似度搜尋。
>
