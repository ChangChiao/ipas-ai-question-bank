# MLOps / LLMOps

> **📝 概念解釋：**
> MLOps (Machine Learning Operations) 是一套結合開發(Dev)和維運(Ops)的文化和實踐法，旨在為機器學習生命週期建立起「自動化且穩定」的標準流水線（包含數據收集、模型追蹤、自動化測試、一鍵佈署與效能監控）。而衍生出的 LLMOps (大型語言模型維運) 則是特別針對維護龐大 LLM 的工具鏈，包含追蹤 Prompt 版本、處理微調成本、與監控模型是否產生幻覺。
>
> **💡 使用場景：**
> 幫助數據科學家和工程團隊擺脫將模型只放在自己筆電跑的困境。透過 MLOps 架構，當收集到新的數據漂移訊號時，系統會自動觸發腳本在背景重訓，並在隔離區測試通過後自動推上線，從而確保 AI 產品品質的一致性。
>
> **🔗 關聯關鍵字：**
> `DevOps`, `CI/CD`, `Model Deployment`, `Model Monitoring`
