# MaaS (模型即服務)

> **📝 概念解釋：**
> 模型即服務 (Model as a Service, MaaS) 是一種雲端運算的商業模式。平台供應商(如 OpenAI, AWS, Google Cloud) 把超級龐大、訓練並維護成本極高的基礎語言/圖片生成模型部署在雲端，然後開放各項 API 網路接口供大眾遠端調用，開發者不需再煩惱去買顯示卡或者架設昂貴的基建。
>
> **💡 使用場景：**
> 各種依靠 API 呼叫 ChatGPT 的第三方中小型應用服務。例如建立一個線上自動生成履歷的網站，網站本身不用具備任何 AI 運行環境，只需按用量將使用者的資訊透過 API 送給 MaaS 供應商處理，再將結果顯示給使用者。
>
> **🔗 關聯關鍵字：**
> `API`, `Cloud Computing`, `SaaS`, `Foundation Models`
