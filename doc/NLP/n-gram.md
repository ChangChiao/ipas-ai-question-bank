# n-gram

> **📝 概念解釋：**
> N-gram 是一種在傳統 NLP 與計算語言學中用來將文本切分成多個連續字詞 (或字元) 組合的模型。「N」代表視窗大小，例如 Unigram 代表單個詞語出現的頻率；Bigram 代表連續兩個詞的組合 (如「機器 / 學習」)；Trigram 就是連續三個字詞的模式。利用 N-gram 統計，模型可以捕捉到局部的字詞搭配機率。
>
> **💡 使用場景：**
> 過去常應用於拼字檢查、關鍵字擷取以及早期的文字生成任務（手機鍵盤的下一個字預測）。也可做為訓練 TF-IDF 時的特徵擴充。
>
