# NLG (自然語言生成)

> **📝 概念解釋：**
> 自然語言生成 (Natural Language Generation, NLG) 是 NLP 的另一個重要分支，與 NLU 相反，它負責「把電腦結構化的資料或內部的處理結果，轉換為人類能讀懂的自然語言」。現代的 NLG 多半藉由強大的深度學習模型來預測下一個字，以生成連貫且合乎語法的長篇文本。
>
> **💡 使用場景：**
> 自動撰寫財經新聞摘要、氣象播報文案、甚至是由 ChatGPT 生成的回覆內容、程式碼註解等，都是 NLG 的實際應用體現。
>
> **🔗 關聯關鍵字：**
> `NLP`, `NLU`, `Text Generation`, `Transformers`
