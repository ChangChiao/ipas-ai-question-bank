# NLP Overview

## 📝 概念解釋 (Concept Explanation)
自然語言處理 (Natural Language Processing, NLP) 是一門結合電腦科學、人工智慧與語言學的跨領域學科。其終極目標是讓電腦能夠「讀懂、理解並生成」人類使用的自然語言（包含文字與語音）。
NLP 是人工智慧走向通用化的關鍵，因為人類擁有全世界最豐富的知識庫都是用文字儲存的。NLP 包含了語言的表層結構分析（如斷詞、詞性標註）到深層語意理解（如推論、情感分析與問答），並隨著近年神經網路與大型語言模型的演進，取得了驚人的突破進展。

## 💡 使用場景 (Use Cases)
- **機器翻譯 (Machine Translation)**：如 Google 翻譯，能快速、流暢地將長篇大論的英文新聞稿自動翻譯成中文或其他數百種語言。
- **虛擬助理與對話機器人**：像 Apple Siri 或各大電服中心的客服機器人，能夠聽懂使用者的自然口語提問，從而自動幫忙訂位、查詢天氣或回答常見問題。
- **自動摘要 (Text Summarization)**：自動將上萬字的會議記錄或長篇財經報告，濃縮提煉成只有三五百字的精華摘要，幫助讀者快速掌握重點。
