# Autoregressive Model (自迴歸模型)

> **📝 概念解釋：**
> 自迴歸模型 (Autoregressive Model, AR Model) 是指一種利用過去的一系列自身觀察值（歷史數據）去預測當前或未來數值的統計模型。在自然語言處理領域中，它被特指為「文字的生成必須由左至右、依賴前面已經生成出來的字，來預測決定下一個字」的生成機制。
>
> **💡 使用場景：**
> 所有如 GPT 家族的語言模型，其底層文本生成方式皆為自迴歸。除了文字生成，也常用在氣象的時間序列預測（利用過去五天的氣溫預測明天氣溫）。
>
