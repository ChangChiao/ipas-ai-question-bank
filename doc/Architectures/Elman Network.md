# Elman Network

> **📝 概念解釋：**
> Elman Network（又稱簡單循環神經網絡 SRN）是早期循環神經網路 (RNN) 的一種基本型態。它的架構特點是在網路隱藏層外圍，加入了一個所謂的「上下文層 (Context Layer)」。這個上下文層會儲存並「回送」前一個時間步驟中隱藏層的狀態，來當作下一次輸入的一部份。這種將內部資訊再利用的設計，讓模型擁有了最初階的「記憶」能力。
>
> **💡 使用場景：**
> 為 LSTM 和現代 RNN 奠定了發展基礎。常用作學術界理解遞迴運算的範例，早期也被用於簡單的時間序列訊號處理(如預測特定波形)和基礎的文法結構學習。
>
> **🔗 關聯關鍵字：**
> `RNN`, `Context Layer`, `Jordan Network`, `Sequence Data`
