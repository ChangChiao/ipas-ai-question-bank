# Deep Learning

## 📝 概念解釋 (Concept Explanation)
深度學習 (Deep Learning) 是機器學習 (Machine Learning) 的一個分支，其靈感來自人類大腦中神經元互相連結的結構，主要由一種被稱為「人工神經網路 (Artificial Neural Networks, ANN)」的模型所組成。
所謂「深層 (Deep)」，指的是該模型擁有許多隱藏層 (Hidden Layers)。傳統機器學習通常依賴人類工程師手工設計特徵 (Feature Engineering) 給模型；而深度學習具備「自動特徵萃取 (Feature Extraction)」的能力，能夠透過一層又一層的神經網路，從原始資料（例如像素、文字字元、音訊波形）中層層堆疊出越來越高階、抽象的特徵。

## 💡 使用場景 (Use Cases)
- **電腦視覺 (Computer Vision)**：使用深度神經網路（如 CNN 卷積神經網路）來精準進行人臉辨識、車牌辨識、腫瘤影像醫學檢測以及自動駕駛車輛的環境感知。
- **自然語言處理 (NLP) 與生成式AI**：Siri、Alexa 語音助理的語音識別，以及 ChatGPT、Gemini 等大型語言模型背後的 Transformer 基礎架構，都是深度學習在語言文字處理上的巨大成就。
- **複雜推薦系統**：Netflix、YouTube 等平台利用深層神經網路模型預測使用者可能喜歡的下一部影片，進而最大化用戶的觀看時長與黏著度。
