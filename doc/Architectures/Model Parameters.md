# Model Parameters

## 📝 概念解釋 (Concept Explanation)
模型參數 (Model Parameters) 是機器學習模型中「可透過訓練資料自動學習並更新的內部變數」。最常見的模型參數包含：神經網路連線上的「權重 (Weights)」和各節點的「偏差值/偏置 (Biases)」。
可以把模型想像成一個巨大的數學公式函數，而這些參數就是決定公式具體樣貌的神秘數字。在訓練過程中，模型會不斷調整這些參數，以縮小預測結果和標準答案之間的誤差 (Loss)。因此，模型的「參數數量」往往代表了該模型的複雜度與學習能力。

## 💡 使用場景 (Use Cases)
- **大型語言模型 (LLM) 的規模指標**：我們常聽到的 Llama-7B, GPT-3 的 175B，裡面的 B 指的就是 Billion (十億)。7B 代表這個模型內部有 70 億個訓練好的參數。參數越多，模型儲存的世界知識和語言理解能力通常越強。
- **權重遷移與微調 (Fine-Tuning)**：當我們使用預訓練模型 (Pre-trained Model) 進行遷移學習時，我們實際上是在下載這數億個已經訓練好的最佳「模型參數」，並在大規模數據基礎上稍微修改少數幾層參數以適應我們特定的新任務。
