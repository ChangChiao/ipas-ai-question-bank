# Prompt Engineering

> **📝 概念解釋：**
> 提示詞工程 (Prompt Engineering) 指的是如何透過精心設計指令與提供上下文範例，來引導並最大化激發出生成式 AI 語言模型的表現潛能。這包含了了解模型的運作特性，並套用不同的系統性溝通框架 (如 Few-shot Prompting 提供少量範例、Chain-of-Thought 要求列出思考過程) 來強制約束模型不產生幻覺並產出精確格式。
>
> **💡 使用場景：**
> 現代開發者必備技能。比如從傳統簡單地要求 `幫我寫一個網頁`，轉化成 `你現在是一位擁有 10 年資歷的資深前端工程師(角色設定)。請幫我撰寫以下規格的 React 元件...並以 JSON 格式輸出(約束輸出)`，以獲得能直接投入生產線的可靠成果。
>
> **🔗 關聯關鍵字：**
> `Chain-of-Thought`, `Few-shot`, `LLM`, `Prompt Injection`
