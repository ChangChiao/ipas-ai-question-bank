# LRM (大型推理模型)

> **📝 概念解釋：**
> 大型推理模型 (Large Reasoning Model, LRM) 指的是專門強化「邏輯推導與複雜思考」的大型語言模型變體（如 OpenAI 的 o1 模型）。不同於強調廣泛知識或文字生成多樣性的傳統 LLM，LRM 內建了思維鏈 (Chain-of-Thought) 的演算法機制，在給出最終答案前，會自動將問題分解成細微步驟，花費更多運算時間 (Test-time Compute) 在內部自我驗證、反思與除錯。
>
> **💡 使用場景：**
> 專注於解決高度複雜且需要多步驟嚴謹證明的任務，例如高階程式碼重構、高等數學證明題、以及藥物分子的設計推理等。這類模型雖然「思考」較慢，但對於數理邏輯考驗的精確度卻能超越傳統 LLM。
>
