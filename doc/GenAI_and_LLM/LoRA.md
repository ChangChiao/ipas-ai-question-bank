# LoRA (低秩自適應)

> **📝 概念解釋：**
> 低秩自適應微調技術 (Low-Rank Adaptation, LoRA) 是由微軟提出的一種極致高效的參數效率微調 (PEFT) 演算法。傳統上，若要微調擁有上千億參數的 LLM 需要極大的算力。LoRA 運用了數學線性代數的「低維度矩陣」概念，在原本神經網路龐大且凍結不變的權重(Weights)旁，外掛上兩個極小的低維矩陣 (Rank) 來間接模擬學習變化。這能讓訓練所需的參數數量縮減 10,000 倍以上，同時保持幾乎相同的模型效能。
>
> **💡 使用場景：**
> 一般開發者與開源社群最常使用的 LLM 微調方式（如使用在 Llama 3 或 Stable Diffusion 的特化訓練）。例如：只用一張消費級顯示卡，在一小時內將通用的繪圖模型微調出專門畫「特定動漫角色」的新模型。
>
> **🔗 關聯關鍵字：**
> `PEFT`, `Fine-tuning`, `LLM`, `Transformer`
