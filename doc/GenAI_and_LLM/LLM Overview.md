# LLM Overview

## 📝 概念解釋 (Concept Explanation)
大型語言模型 (Large Language Models, LLM) 是近年人工智慧最具突破性的發展，屬於生成式 AI 的核心基礎。它是一種規模極度龐大的深度學習模型（通常具備數十億到數千億個參數，並建立在 Transformer 架構之上），透過閱讀和學習網際網路上近乎全人類的海量文字資料庫來進行訓練。
LLM 的核心運作原理其實非常單純——它在做「文字接龍」。根據你輸入的提示詞 (Prompt)，它可以非常精準地計算並預測出下一個最合理的單字是什麼。因為訓練資料包羅萬象，這使得它看起來像是具備了類似人類的「理解能力」與「常識」，可以幫忙寫程式、寫文章、翻譯甚至進行邏輯推理。

## 💡 使用場景 (Use Cases)
- **萬能虛擬助手**：如 ChatGPT、Claude 等對話式 AI，能回答你在生活中遇到的百般雜項問題，從幫小學生解數學題，到幫忙規劃五天四夜的日本東京自由行行程。
- **程式碼輔助開發 (Copilot)**：軟體工程師只需要寫下一句註解 "寫一個能連接資料庫並抓取用戶資料的 Python 函數"，LLM 就能在幾秒內自動把整段完整的後端程式碼寫出來。
- **自動化客服與客製信件**：企業將 LLM 整合到網站客服中，用非常禮貌且專業的口吻應對客人的退換貨請求，並且不會像傳統罐頭訊息那般死板。
